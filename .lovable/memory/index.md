# Project Memory

## Core
MakoVote: broadcast polling graphics system for live sports/TV. Dark UI, Inter + JetBrains Mono, orange accent (hsl 24 95% 53%).
Three surfaces: Operator Control, Program Display (1080p), Viewer Voting. Not a consumer app — broadcast-grade tool.
Mako Systems brand family. Shark fin logo asset at src/assets/mako-illustration.png.
Draft preview area MUST stay lifted: pt-2 px-4 pb-4, gap-2, justify-start (not centered). Do not revert.
